🍎 iOS / 아이폰 호환성 수정
  아이폰 사파리에서 무한 로딩되던 문제를 해결했어요. CSS inset 속성과 JavaScript optional chaining(?.) 문법을 구버전 브라우저에서도 동작하도록 변환했어요.

🔧 기타 버그 수정

  직업 필터 선택 시 해당 직업의 부캐도 함께 표시되며, 구분선으로 본캐/부캐를 구분해서 보여줘요.
  파티 모집 종류를 원정/초월/성역/어비스(하층)/어비스(중층)/투기장/토벌전으로 세분화했어요.
패치노트 v1.2.0 — 파티 모집 강화 · 로그인 · 공지 시스템
2026.03.25
👤 닉네임 로그인 시스템
  이제 닉네임으로 로그인하면 내 정보를 한눈에 볼 수 있어요. 내가 올린 모집글, 신청 내역, 댓글도 한 곳에서 확인 가능해요. 군단장/엘리트장교는 모든 글과 댓글을 관리할 수 있어요.

⚔ 파티 모집 대폭 개선
  모집 종류가 원정/초월/성역/어비스(하층)/어비스(중층)/투기장/토벌전으로 세분화됐어요. 제목 검색과 기간 필터도 생겼고, 신청자 닉네임을 입력할 때 멤버 목록에서 바로 검색할 수 있어요. 모집글 작성자는 글 수정도 가능해요.

📋 공지/패치노트 탭 추가
  레기온 공지사항과 패치노트를 이 탭에서 확인할 수 있어요. 군단장/엘리트장교는 공지를 직접 작성할 수 있어요.

🔔 슈고 타이머 개선
  매 시 15분, 45분에 TTS로 "슈고 시간입니다" 알림과 브라우저 알림을 받을 수 있어요. 볼륨 조절도 가능해요.
패치노트 v1.1.0 — 스펙 상세 · 장비 · 어비스 급수
2026.03.25
🛡 캐릭터 상세 스펙 개편
  NC 공홈처럼 무기/방어구, 장신구/룬, 아르카나를 한 화면에서 볼 수 있어요. 장비 클릭하면 세부 옵션도 확인 가능해요. 스탯은 기본 6개 + 더보기로 전체를 볼 수 있어요.

🎖 어비스 급수 표시
  멤버 스펙 목록에 어비스 급수가 표시되고, 급수 기준으로 정렬할 수 있어요. (훈련병 ~ 총사령관)

⚙ 본캐/부캐 시스템
  부캐 등록 시 본캐와 묶어서 관리할 수 있어요. 멤버 스펙에서 "부캐 보기" 버튼으로 펼쳐볼 수 있어요.

📊 멤버 스펙 정렬 기능
  닉네임, 직업, 계급, 전투력, 아이템레벨, 어비스 급수로 오름/내림차순 정렬이 가능해요.
